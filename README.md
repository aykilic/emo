# Alış Veriş Sitesi ve Yönetim Paneli

Varyantlı ürün satılabilen alış veriş sitesi.
İşlemler web socket hizmeti ile yönetim tarafına anlık olarak bildirilir.

## Kullanılan Teknolojiler

  ### Backend
    * nodejs
    * expresjs
    * apollo server
    * graphql
    * sequelize
    * mongodb
  ### Frontend
    * vuejs
    * vuex(State management)
    * quasar framework
